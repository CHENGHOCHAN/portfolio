SYSTEM ANALYSIS · 企劃設計

觀察地形，而不是正面硬碰

《巨人與孤獨星球》把巨大敵人的壓迫感轉化成路線判斷。玩家要閱讀地形、機關與巨人行為，找出能逃脫或反制的路徑，而不是只靠連續攻擊解決問題。

DESIGN THESIS在舒適明亮的自然景色中，讓孤身旅行者面對 10：1 尺度的泰坦巨人；孤獨不只來自荒涼，也來自世界過於遼闊而只剩自己前進。
01探索遺跡

閱讀道路、地標與傳送門位置。

02辨識威脅

觀察巨人、陷阱與高度差。

03啟動機關

運用跳板、壓力板與大炮。

04逃脫或反制

選擇跑酷路線並規避傷害。

05前往下一區

用傳送寶石解鎖新的遺跡。

SYSTEM 01 · SCALE

巨人尺度與威脅閱讀

泰坦巨人是資源戰爭遺留的自動化兵器。巨人與主角約 10：1 的比例，讓玩家先感受到壓迫，再從動作與場地尋找安全窗口。

泰坦巨人10：1 比例地形判讀
SYSTEM 02 · TRAVERSAL

機關解謎與垂直移動

彈跳板是最重要的移動機關，配合充能壓力板、大炮壓力板、尖刺與跑酷平台，把解謎、移動和逃脫整合成同一段流程。

SYSTEM 03 · WORLD

傳送寶石與世界導航

旅行者幼年取得傳送寶石，因而能解鎖星球各地的傳送門。它同時是劇情道具與導航權限，引導玩家前往最終目的地「四季浮島」。

旅行者傳送寶石四季浮島
SYSTEM 04 · IMMERSION

動態環境與材質音效

風向循環掃動讓草地與樹林產生自然擺動；走路、跑步、翻滾與劈砍則依地面或碰撞材質提供不同聲音，補足世界的觸感。
