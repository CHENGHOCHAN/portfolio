# 莊河泉｜遊戲企劃作品集網站

這個倉庫用來整理我的互動式作品集網站內容。網站主題以像素遊戲場景、載入動畫、作品庫、開發日誌、個人介紹、聯絡頁與履歷下載為核心。

## 網站內容

- 首頁：像素風場景、角色行走與怪物循環動畫、作品入口與最新開發動態。
- 作品庫：以《巨人與孤獨星球》與《異界之魂》為主，呈現實機影片、系統解析、角色、怪物、場景、UI、機關與道具設計。
- 開發日誌：整理學習歷程、製作進度、校內展、新一代設計展與放視大賞參展紀錄。
- 關於我：介紹莊河泉的遊戲企劃方法、技能與履歷下載。
- 聯絡頁：提供 Gmail 與 Instagram 聯絡方式。
