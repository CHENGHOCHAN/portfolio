MY CONTRIBUTION

我負責把魂系參考轉成可討論的戰鬥規則

此專案中我擔任遊戲總監，將世界觀、戰鬥節奏、Boss 階段、道具功能與 UI 顯示整理成團隊能分工製作的規格。

  • 戰鬥設計拆解格擋、閃避、反擊、精力消耗與死亡重試，建立玩家從失敗中學習敵人節奏的核心循環。
  • Boss 節奏規劃監獄長與三階段壓力曲線，讓攻擊前搖、距離判斷與資源保留成為主要挑戰。
  • 成果整理將 72 項素材整理為角色、Boss、場景、道具、UI、建模與企劃文件，強化作品頁的閱讀順序與設計依據。

SYSTEM ANALYSIS · 企劃設計

失敗不是結束，而是讀懂敵人的方法

《異界之魂》以精力管理、翻滾閃避、雙手武器與目標鎖定建立硬核戰鬥。玩家從死亡中學習攻擊節奏，再利用存檔點、道具與靈魂碎片逐步深入地牢。

DESIGN THESIS玩家扮演受到詛咒的不死者，從邊境監獄逃離，收集帝國之主散落各地的靈魂碎片；每一次戰鬥都同時推進能力成長與世界真相。
01探索地牢

尋找道路、存檔點與關鍵道具。

02鎖定敵人

辨識攻擊前搖與安全距離。

03管理精力

在攻擊、翻滾與奔跑間取捨。

04死亡與學習

重生並調整下一次戰術。

05奪回碎片

強化自身並開啟逃離路線。

SYSTEM 01 · COMBAT

精力驅動的雙武器戰鬥

生命與精力是核心資源。左、右武器攻擊、翻滾、奔跑和目標鎖定共同形成風險判斷，玩家必須保留足夠精力應對下一次攻擊。

HP / Stamina雙手武器目標鎖定翻滾
SYSTEM 02 · DEATH

死亡代價與葬堆存檔點

死亡可以重生，但必須付出代價；「死亡骷髏」能把玩家送回最近的葬堆。紅色 YOU DEAD 畫面則明確結束當次嘗試並促使玩家重新判讀。

重生代價死亡骷髏葬堆YOU DEAD
SYSTEM 03 · WORLD

異域大陸與靈魂碎片

神族政權被人、鬼、龍三族推翻後，帝國之主遭到刺殺，身軀化為靈魂碎片。主角從監獄取得碎片後，踏上恢復自身並重整大陸的旅程。

不死者三族衝突靈魂碎片騎士證明
SYSTEM 04 · UI

戰鬥資訊與功能分頁

HUD 顯示生命、精力、雙手武器、裝備道具與異域幣；裝備、背包、地圖和設定則拆分為獨立選單，載入畫面補充操作與世界資訊。
